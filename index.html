Code Block

Here is the famous "hello, world" program written in C: 

#include <stdio.h>

int main()
{
    printf("hello, world\n");
    return 0;
}

On a Unix or Unix-like system, this program can be compiled to a binary executable named a.out with the cc command as shown below. 

$ cc hello.c && ./a.out
hello, world
